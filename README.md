# NikeStore
Nike app is a shoe store app. This app developed with Kotlin and MVVM architecture and application that works with Real rest API.

## This Project Use:

-MVVM Architecture

-Navigation Component

-Dagger-Hilt Dependency Injection

-Retrofit

-Fresco

-Coroutines And Flow

-Room DB

-Timber

-Logging Interceptor

-EventBus
